Slicer files for Prusa Slicer, containing the geometry and modifiers. 
They are optimized for a 0.3 mm nozzle. Printing with 0.4 should also work, but might yield heavier parts. 
Check the extrusion width settings in the "Advanced" print settings tab, external perimeters are set to 0.35 mm for some parts.
If no "L" files are available, simply mirror the parts. Caution: Wing 1 L and R are not symmetric, cable channels are different.
Most parts should fit in a 200x200x200 printer, if not, just split in your slicer.
